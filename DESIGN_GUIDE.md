# fucuu ブランドデザイン・ガイドライン

このドキュメントは、fucuuブランドサイトのビジュアル一貫性を保つための設計図および運用フローです。

## 1. デザイン原則
- **温もりのある余白**: 忙しい日常の中に「余白」を感じさせるため、大きなパディングと控えめな文字配置を徹底します。
- **有機的な形状（もこもこ）**: ロゴ由来の柔らかい形状を、アイコン背景や装飾要素として一貫して使用します。
- **日本語主役のタイポグラフィ**: 英語（セリフ体）はあくまで情緒的な添え物とし、情報は日本語（Noto Serif/Sans JP）で明確に伝えます。

## 2. 資産（アセット）定義
ファイルの取り違えを防ぐため、以下のマッピングを厳守してください。

- **ブランドロゴ (Logo)**: `assets/brand_character.png`
  - 内容: "fucuu" の文字が入った、もこもこしたblob形状のロゴ。
  - 用途: ナビゲーションバー、ヒーローセクション（最大幅320px）、フッター。
- **ブランドキャラクター (Character)**: `assets/brand_logo.png`
  - 内容: 梟（フクロウ）のイラストと "fucuu" の文字の組み合わせ。
  - 用途: 生薬説明などのガイド要素、ユーザーへの語りかけセクション。

## 3. CSS 実装仕様
### もこもこ形状 (Mokomoko Shape)
アイコン背景などに使用する基本形状です。
```css
border-radius: 60% 40% 70% 30% / 50% 60% 40% 50%;
border: 1px solid rgba(142, 151, 117, 0.2);
background: rgba(142, 151, 117, 0.1);
```

### ヒーローセクション
- センターレイアウトを採用。
- 巨大なロゴ（`brand_character.png`）を中央上部に配置。
- 背景に極めて薄い（opacity 0.05程度）大きなもこもこ形状を装飾として配置し、空間の寂しさを解消します。

### 固定ナビゲーション
- **構成**: `[ハンバーガー + 左ロゴ] --- [小ナビ(PC) + Shop Now]`
- **表示仕様**:
    - **左側配置への変更**: テキストリンク（About等）との重なりを避けるため、ロゴを左側（ハンバーガーメニューの隣）へ移動。
    - **視認性向上**: ロゴ背景の白枠を解消するため `mix-blend-mode: multiply` を適用。
    - **不要なリストの排除**: モバイルメニューがPC版で漏れ出さないよう、`nav-links-mobile` は常に `display: none`（アクティブ時のみモバイルで表示）を徹底。
- **高さ管理**: `min-height: 70px` を確保。



## 4. コンテンツ・バランス
- **ヒーロー見出し**: 「心がフクフクする、」の後に改行を入れ、視覚的な安定を図る。
- **セクション余白**: 生薬図鑑など、リスト形式のコンテンツはタイトルから離れすぎないよう、`margin-top` を `var(--space-m)` (2rem) 程度に抑える。
- **キャラクターの台詞**: `character-voice` クラスの吹き出しレイアウトを使用し、三人称ではなくキャラクターが直接話している口調とする。
