# fucuu ブランドサイト

「心がフクフクする、」
忙しい日常の中に温もりのある余白を提案する、バスソルト & アロマブランド「fucuu」の公式サイトです。

## 概要
このプロジェクトは、Amazonで展開しているfucuuブランドの世界観をより深く伝えるためのライフスタイル型ブランドサイトです。生薬の知識やブランド哲学を、親しみやすいデザインとタイポグラフィで表現しています。

## デザイン原則
- **温もりのある余白**: 日常の忙しさを忘れさせる、ゆったりとしたレイアウト。
- **もこもこ（Organic Shapes）**: ブランドのアイコンである柔らかい形状を各所に使用。
- **日本語主役**: 情報は日本語で明確に伝えつつ、セリフ体の英語で情緒的なアクセントを加えています。

## 構成
- `index.html`: メインページ
- `index.css`: デザインシステム、アニメーション、レスポンシブ設定
- `assets/`: ブランドロゴ、キャラクター画像、生薬素材などのアセット
- `DESIGN_GUIDE.md`: 詳細なデザイン・アイデンティティ運用ガイド

## 開発
このサイトはバニラHTML/CSSで構築されており、シンプルかつ高速な読み込みを重視しています。
Google Fonts（Inter, Noto Serif JP, Noto Sans JP）を使用しています。
